# David Medina Personal Portfolio Website

Este repositorio contiene el código fuente del sitio web de portafolio personal de David Medina. El objetivo es crear un sitio web profesional de una sola página, inicialmente en español, utilizando la plantilla `Koyta-two`.

## Estado Actual del Proyecto

El proyecto se encuentra en una fase avanzada de integración de contenido para la versión en español. Se han completado las siguientes secciones:

*   **Configuración Inicial:** El entorno de desarrollo está configurado y la estructura de archivos de entrada (`Input/DME`) ha sido organizada.
*   **Página de Inicio (Hero Section):** Personalizada con el nombre, roles profesionales (con animación de texto) y foto de perfil de David Medina.
*   **Sobre Mí:** Sección actualizada con la biografía de David Medina.
*   **Servicios y Habilidades:** Secciones pobladas con los servicios y habilidades clave.
*   **Contacto y Pie de página:** Información de contacto y enlaces a redes sociales actualizados.
*   **Selector de Tema Temporal:** Se han añadido botones temporales para probar diferentes esquemas de color en el sitio.

## Características Clave

*   Diseño de una sola página.
*   Diseño responsivo.
*   Animación de texto dinámico en la sección principal.
*   Integración de contenido en español.
*   Selector de tema interactivo (temporal para demostración).

## Tecnologías Utilizadas

*   HTML5
*   CSS3 (SASS para preprocesamiento)
*   JavaScript

## Uso Local

Para ver el sitio web localmente:

1.  Clona este repositorio: `git clone https://github.com/planckc/davidmedinap2.git`
2.  Navega al directorio del proyecto: `cd davidmedinap2`
3.  Abre el archivo `david_medina_website/index.html` en tu navegador web preferido.
